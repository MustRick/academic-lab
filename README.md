# Web Sitesi 2

React frontend ve Express.js backend iceren basit tam yigin proje.

## Kurulum

```bash
npm install
```

## Gelistirme

```bash
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:5001
- Ornek API: http://localhost:5001/api/health

## Scriptler

```bash
npm run build
npm start
npm run lint
```
